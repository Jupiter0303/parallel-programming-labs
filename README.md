# Лабораторная работа №1

## Задание
* Написать программу на языке C/C++ для перемножения двух квадратных матриц.
* Исходные данные: файлы, содержащие значения исходных матриц.
* Выходные данные: файл со значениями результирующей матрицы, время выполнения, объем задачи.
* Обязательна автоматизированная верификация результатов вычислений с помощью Python (NumPy).

---

## Алгоритмические особенности
В проекте реализована оптимизированная схема перестановки циклов **i-k-j**. В отличие от стандартного подхода (i-j-k), этот метод значительно повышает эффективность работы с кэш-памятью процессора.

**Преимущества порядка i-k-j:**
* **Локальность данных:** Обращение к памяти происходит последовательно, что минимизирует промахи кэша.
* **Скорость:** Линейный доступ позволяет процессору задействовать механизмы предвыборки данных.

## ✅ Верификация и контроль точности
Для проверки корректности используется Python-скрипт verify.py:
* **Эталон:** Результаты C++ сопоставляются с функцией numpy.dot().
* **Точность:** Сравнение через numpy.allclose() с допуском atol=1e-1.
* **Автоматизация:** Скрипт генерирует случайные данные, запускает программу и выводит вердикт.

---

## Инструкция по запуску
1. Скомпилировать: `g++ -O3 src/main.cpp -o matmul.exe`
2. Запустить тесты: `python scripts/verify.py`
